API Reference¶

GeoExt components and data utility cl***REMOVED***es extend map related functionality to equivalent cl***REMOVED***es in Ext. The API reference here documents the properties, methods, and events that are extensions or modifications to the Ext parent cl***REMOVED***es. Documentation for each cl***REMOVED*** contains links to the Ext parent cl***REMOVED***, and for a full picture of the API, it is essential to have a copy of the Ext API Documentation at hand.

GeoExt cl***REMOVED***es are typically configured with OpenLayers objects. For detail on methods and properties provided by these objects, see the OpenLayers API Documentation.
