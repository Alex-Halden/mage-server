GeoExt.data.WMSCapabilitiesReader¶

Extends
  • Ext.data.DataReader
cl***REMOVED*** GeoExt.data.WMSCapabilitiesReader(meta, recordType)¶
Parameters:
  • meta – Object Reader configuration from which: layerOptions is an optional object p***REMOVED***ed as default options to the OpenLayers.Layer.WMS constructor. layerParams is an optional set of parameters to p***REMOVED*** into the OpenLayers.Layer.WMS constructor.
  • recordType – Array | Ext.data.Record An array of field configuration objects or a record object. Default is GeoExt.data.LayerRecord with the following fields: name, title, abstract, queryable, opaque, noSubsets, cascaded, fixedWidth, fixedHeight, minScale, maxScale, prefix, formats, styles, srs, dimensions, bbox, llbbox, attribution, keywords, identifiers, authorityURLs, metadataURLs, infoFormats. The type of these fields is the same as for the matching fields in the object returned from OpenLayers.Format.WMSCapabilities::read().

Data reader cl***REMOVED*** to create an array of GeoExt.data.LayerRecord objects from a WMS GetCapabilities response.

Configuration properties in addition to those listed for Ext.data.DataReader.

attributionCls
String CSS cl***REMOVED*** name for the attribution DOM elements. Element cl***REMOVED*** names append “-link”, “-image”, and “-title” as appropriate. Default is “gx-attribution”.
