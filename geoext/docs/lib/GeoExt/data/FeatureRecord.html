GeoExt.data.FeatureRecord¶

Extends
  • Ext.data.Record
cl***REMOVED*** GeoExt.data.FeatureRecord¶

A record that represents an OpenLayers.Feature.Vector. This record will always have at least the following fields:

  • state String
  • fid String
