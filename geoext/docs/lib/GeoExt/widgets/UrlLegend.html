GeoExt.UrlLegend¶

Extends
  • Ext.Container
  • GeoExt.LayerLegend
xtype
gx_urllegend
cl***REMOVED*** GeoExt.UrlLegend(config)¶
Show a legend image in a BoxComponent and make sure load errors are dealt with.

Config Options¶

Configuration properties in addition to those listed for Ext.Container.

labelCls
String Optional css cl***REMOVED*** to use for the layer title labels.
layerRecord
GeoExt.data.LayerRecord The layer record for the legend
legendTitle
String Optional title to be displayed instead of the layer title. If this is set, the value of showTitle will be ignored (***REMOVED***umed to be true).
showTitle
Boolean Whether or not to show the title of a layer. This can be overridden on the LayerStore record using the hideTitle property.
