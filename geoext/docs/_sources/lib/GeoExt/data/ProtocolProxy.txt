Config Options
--------------

Configuration properties in addition to
those listed for `Ext.data.DataProxy <http://dev.sencha.com/deploy/dev/docs/?cl***REMOVED***=Ext.data.DataProxy>`_.


.. describe:: abortPrevious

    ``Boolean``
    Abort any previous request before issuing another.  Default is ``true``.

.. describe:: protocol

    ``OpenLayers.Protocol``
    The protocol used to fetch features.

.. describe:: setParamsAsOptions

    ``Boolean``
    Should options.params be set directly on options before p***REMOVED***ing it into
    the protocol's read method? Default is ``false``.
