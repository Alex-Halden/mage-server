:mod:`GeoExt.plugins`
=====================

The :mod:`GeoExt.plugins` module contains cl***REMOVED***es that add functionality to
GeoExt components like :cl***REMOVED***:`GeoExt.MapPanel`, or connect form fields to
:mod:`GeoExt.data` cl***REMOVED***es.
