About {{name}}

Mobile applications provide situational awareness capabilities to disconnected and disadvantaged users. MAGE enables imagery sharing, georeferencing, tracking, and communication between users supporting mobile and desktop users.

Designed to merge the best of breed features from MFO and Hermes based on GIAT technical experience and community feedback. MAGE delivers an enhanced mobile situational awareness capability for a broad community of users.

Tech Stack

The MAGE server is built on NodeJS, with data stored in MongoDB via mongoose. Routing provided with express, and authentication with passport.

The front end maps are powered by Leaflet. The responsive layouts and buttons are styled by a modified version of Bootstrap. The whole front end is tied together by AngularJS.
